1) Установить MySQL сервер
2) Создать новую БД через MySQL Command Line, введя команду:
    CREATE DATABASE crud;
3) В файле crud/crud/settings.py настроить подключение к БД, изменив значение переменной DATABASES:
    DATABASES = {
        'default': {
            'ENGINE': 'django.db.backends.mysql',
            'NAME': '<имя_созданной_бд>',
            'USER' : '<имя_пользователя_для_бд>',
            'PASSWORD' : '<пароль_для_этого_пользователя>',
            'HOST' : 'localhost'
        }
    }
4) Открываем localhost:8000 и взимодействуем с API через веб-интерфейс :)


* Примечание *
При создании и изменении юзера - введенный пароль валидируется. Требования:
  1) Больше 8 символов.
  2) Не должен быть в списке распространенных паролей. Этот список находится в файле, common-passwords.txt.gz . Он содержит 		  список примерно из 20000 распространенных паролей.
  3) Не должен содержать только лишь одни цифры.
